# Random Number Guessing Game

A simple Python command-line game where the computer generates a random number and the player tries to guess it. The program gives feedback after each guess until the correct number is found, then offers the chance to play again.

## 🚀 Features

- Generates a random number between 1 and 100
- Accepts user input for guesses
- Provides hints:
  - 📈 Too High
  - 📉 Too Low
- Shows the number of attempts
- Lets the player play again after winning
- Beginner-friendly Python project

## 🛠️ Technologies Used

- Python 3
- `random` module
